Este codigo permite al usuario encender y apagar unas entradas digitales y hacer una lectura de estas. El usuario mediante unos botones puede apagar y encender los LEDs.

Hardware

  • Arduino Leonardo
  • 4x Resistencia 330 ohm
  • 4x LED
